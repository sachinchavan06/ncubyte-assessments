# Gmail Compose Function Test Plan

This repository contains the manual test plan and cases for testing the **Compose and Send** functionality in Gmail.

- **Traditional Test Cases:** 
A spreadsheet (`GmailComposeTestCases.xlsx`) listing detailed manual test cases (preconditions, steps, expected results, etc.).  
- **BDD Scenarios:** 
A Gherkin feature file (`compose_email.feature`) with Given/When/Then scenarios for key compose/send behaviors.
